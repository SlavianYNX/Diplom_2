1. создать уникального пользователя;

2. создать пользователя, который уже зарегистрирован;

3. создать пользователя и не заполнить одно из обязательных полей.

**Логин пользователя:**

4. вход под существующим пользователем;

5. вход с неверным логином и паролем.

**Создание заказа:**

6. с авторизацией;
7. без авторизации;
8. с ингредиентами;
9. без ингредиентов;
10. с неверным хешем ингредиентов.
